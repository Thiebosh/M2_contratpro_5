# Exécution du service
```
SET HOST=0.0.0.0
SET PORT=8002

SET IS_LOCAL=true

SET CPP_PATH=src/brique2/cpp/
SET DRIVE_PATH=credentials/service_account.json

SET MONGO_URL=***
SET MONGO_USERNAME=***
SET MONGO_PASSWORD=***

python src
```



# Connexion au service
Le serveur est consultable à l'url suivante : http://localhost:8002.
